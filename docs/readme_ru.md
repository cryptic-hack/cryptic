# Карта статистики GitHub

Скопируйте и вставьте это в свой контент markdown, и все. Просто!

Измените значение `?username=` на ваше имя пользователя на Github.

```md
[![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
```

_ Примечание: Доступные ранги: S+ (топ-1%), S (топ-25%), A++ (топ-45%), A+ (топ-60%) и B+ (все).
Значения рассчитываются с помощью [функции кумулятивного распределения](https://en.wikipedia.org/wiki/Cumulative_distribution_function) с использованием коммитов, вкладов, проблем, звездочек, запросов на извлечение, подписчиков и принадлежащих репозиториев.
Реализация может быть исследована на [src/calculateRank.js ](./src/calculateRank.js)._

### Скрытие индивидуальной статистики

Чтобы скрыть какую-либо конкретную статистику, вы можете передать параметр запроса `&hide=` со значениями, разделенными запятыми.

> Параметры: `&hide=stars,commits,prs,issues,contribs`

```md
![Anurag's GitHub stats](https://github-readme-stats.vercel.app/api?username=anuraghazra&hide=contribs,prs)
```

### Добавление количества частных взносов к общему количеству коммитов

Вы можете добавить количество всех ваших личных вкладов к общему количеству коммитов, используя параметр запроса `&count_private=true`.

_ Примечание: Если вы развертываете этот проект самостоятельно, частные взносы будут учитываться по умолчанию. В противном случае вам необходимо выбрать совместное использование ваших личных подсчетов взносов._

> Параметры:`&count_private=true`

```md
![Статистика Anurag на GitHub](https://github-readme-stats.vercel.app/api?username=anuraghazra&count_private=true)
```

### Отображение значков

Чтобы включить значки, вы можете передать `show_icons=true` в параметре запроса, например:

```md
![Статистика Anurag на GitHub](https://github-readme-stats.vercel.app/api?username=anuraghazra&show_icons=true)
```

### Темы

С помощью встроенных тем вы можете настроить внешний вид карты, не выполняя никаких действий [ручная настройка] (#настройка).

Используйте параметр `&theme=THEME_NAME` следующим образом :-

```md
![Статистика Anurag на GitHub](https://github-readme-stats.vercel.app/api?username=anuraghazra&show_icons=true&theme=radical)
```

#### Все встроенные темы:-

dark, radical, merko, gruvbox, tokyonight, onedark, cobalt, synthwave, highcontrast, dracula

Вы можете просмотреть предварительный просмотр [всех доступных тем](./themes/README.md) или проверить [файл конфигурации темы](./themes/index.js ) и ** вы также можете добавить новые темы **, если хотите :D

### Кастомизация

Вы можете настроить внешний вид своей "Карточки статистики" или "Карточки репозитория" по своему усмотрению с помощью параметров URL.

#### Общие параметры:

- `title_color` - Цвет заголовка карточки _(hex color)_
- `text_color` - Цвет основного текста _(hex color)_
- `icon_color` - Цвет значков, если таковой имеется _(hex color)_
- `border_color` - Цвет границы карты _(hex color)_. (Не применяется, если включен параметр `hide_border`)
- `bg_color` - Цвет фона карты _(hex color)_ **или** градиент в виде _angle,start,end_
- `hide_border` - Скрывает границу карты _(boolean)_
- `theme` - название темы, выберите из [все доступные темы](./themes/README.md)
- `cache_seconds` - установите заголовок кэша вручную _(min: 1800, max: 86400)_
- `locale` - установите язык в карточке _ (например, cn, de, es и т.д.)_
- `border_radius` - Закругление угла на карточке_

##### Градиент в bg_color

Вы можете указать несколько значений, разделенных запятыми, в параметре bg_color для отображения градиента в следующем формате:

```
&bg_color=DEG,COLOR1,COLOR2,COLOR3...COLOR10
```

> Примечание о кэше: Кэш карт repo по умолчанию составляет 4 часа (14400 секунд), если количество форков и звездочек меньше 1 кб, в противном случае это 2 часа (7200 секунд). Кроме того, обратите внимание, что кэш ограничен минимум 2 часами и максимум 24 часами.

#### Эксклюзивные опции Карты Статистики:

- `hide` - Скрывает указанные элементы из статистики _(Comma-separated values)_
- `hide_title` - _(boolean)_
- `hide_rank` - _(boolean)_ скрывает ранг и автоматически изменяет ширину карты
- `show_icons` - _(boolean)_
- `include_all_commits` - Подсчитывайте общее количество коммитов, а не только коммиты текущего года _(boolean)_
- `count_private` - Подсчитывать частные коммиты _(boolean)_
- `line_height` - Задает высоту строки между текстом _(number)_
- `custom_title` - Задает пользовательский заголовок для карточки
- `disable_animations` - Отключает все анимации на карте _(boolean)_

#### Эксклюзивные Опции Карты repo:

- `show_owner` - Показать имя владельца репозитория _(boolean)_

#### Эксклюзивные опции Языковой Карты:

- `hide` - Скрыть указанные языки с карты _(Comma-separated values)_
- `hide_title` - _(boolean)_
- `layout` - Переключение между двумя доступными макетами `default` и `compact`
- `card_width` - Установите ширину карты вручную _(number)_
- `langs_count` - Показывать на карте больше языков в диапазоне от 1 до 10, по умолчанию 5 _(number)_
- `exclude_repo` - Исключить указанные репозитории _(Comma-separated values)_
- `custom_title` - Задает пользовательский заголовок для карточки

> :предупреждение: **Важно:**
> Имена языков должны быть экранированы по uri, как указано в [Процентная кодировка](https://en.wikipedia.org/wiki/Percent-encoding )
> (т.е.: `c++` должен стать `c%2B%2B`, `jupyter notebook` должен стать `jupyter%20notebook` и т.д.) Вы можете использовать
> [urlencoder.org ](https://www.urlencoder.org /), чтобы помочь вам сделать это автоматически.

#### Эксклюзивные опции карты Wakatime:

- `hide` - Скрыть указанные языки с карты _(Comma-separated values)_
- `hide_title` - _(boolean)_
- `line_height` - Задает высоту строки между текстом _(number)_
- `hide_progress` - Скрывает индикатор выполнения и процент _(boolean)_
- `custom_title` - Задает пользовательский заголовок для карточки
- `layout` - Переключение между двумя доступными макетами `default` и `compact`
- `langs_count` - Ограничение количества языков на карте, по умолчанию используются все указанные языки
- `api_domain` - Установите пользовательский домен API для карты, например, для использования таких сервисов, как [Hakatime](https://github.com/mujx/hakatime ) или [Wakapi](https://github.com/muety/wakapi )
- `range` – Запросите диапазон, отличный от вашего WakaTime по умолчанию, например, `last_7_days`. См. [Документацию WakaTime API](https://wakatime.com/developers#stats ) для получения списка доступных опций.

# Карта лучших языков

В карточке лучших языков отображается наиболее часто используемый верхний язык пользователя GitHub

_ ПРИМЕЧАНИЕ: Лучшие языки не указывает на мой уровень квалификации или что-то в этом роде; это показатель GitHub, позволяющий определить, на каких языках больше всего кода на GitHub. Это новая функция github-readme-stats._

### Использование

Скопируйте и вставьте этот код в свой readme и измените ссылки.

Конечная точка: `api/top-langs?username=anuraghazra`

```md
[![Лучшие Языки](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra)](https://github.com/anuraghazra/github-readme-stats)
```

### Исключить определенные репозитории

Вы можете использовать параметр `&exclude_repo=repo1,repo2`, чтобы исключить отдельные репозитории.

```md
[![Лучшие Языки](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&exclude_repo=github-readme-stats,anuraghazra.github.io)](https://github.com/anuraghazra/github-readme-stats)
```

### Скрыть отдельные языки

Вы можете использовать параметр `&hide=language1,language2`, чтобы скрыть отдельные языки.

```md
[![Лучшие Языки](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&hide=javascript,html)](https://github.com/anuraghazra/github-readme-stats)
```

### Показать больше языков

Вы можете использовать опцию `&langs_count=`, чтобы увеличить или уменьшить количество языков, отображаемых на карточке. Допустимыми значениями являются целые числа от 1 до 10 (включительно), а значение по умолчанию равно 5.

```md
[![Лучшие Языки](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&langs_count=8)](https://github.com/anuraghazra/github-readme-stats)
```

### Компактная Компоновка Языковой Карты

Вы можете использовать опцию `&layout=compact`, чтобы изменить дизайн карты.

```md
[![Лучшие Языки](https://github-readme-stats.vercel.app/api/top-langs/?username=anuraghazra&layout=compact)](https://github.com/anuraghazra/github-readme-stats)
```

#### [Ознакомьтесь С Пошаговым Видеоуроком От @codeSTACKr](https://youtu.be/n6d4KHSKqGk?t=107)

Поскольку API GitHub допускает только 5 тыс. запросов в час, мой `https://github-readme-stats.vercel.app/api - возможно, это может привести к срабатыванию ограничителя скорости. Если вы размещаете его на своем собственном сервере Vercel, то вам не нужно ни о чем беспокоиться. Нажмите на кнопку развертывания, чтобы начать!
ПРИМЕЧАНИЕ: Начиная с [#58](https://github.com/anuraghazra/github-readme-stats/pull/58 ) мы должны быть в состоянии обрабатывать более 5 тыс. запросов и не иметь проблем с простоями : D
